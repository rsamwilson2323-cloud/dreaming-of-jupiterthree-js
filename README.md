# 🪐 Dreaming of Jupiter

**An Immersive 3D Space Experience Built with Three.js**

🚀 Real-time rendering • 🌙 Orbiting moon • ⭐ Dynamic starfield • 🌌 Cinematic space visuals

Dreaming of Jupiter is a browser-based 3D space simulation built using **Three.js and WebGL**. The project creates a visually stunning cosmic environment featuring a rotating Jupiter-like planet, orbiting moon, animated stars, dynamic terrain effects, and smooth camera movement.

Designed for learning **Three.js, WebGL, 3D graphics, and interactive visual experiences using JavaScript**.

---

# ✨ Features

🪐 **Realistic Planet Rendering**
A detailed Jupiter-inspired planet with realistic textures and smooth rotation.

🌙 **Orbiting Moon Animation**
A dynamically animated moon orbiting around the planet.

⭐ **Animated Starfield**
Thousands of moving stars create a deep-space atmosphere.

🌌 **Space Environment**
Immersive cosmic background with realistic depth and fog effects.

🎥 **Cinematic Camera Motion**
Smooth camera movement for a professional visual experience.

🖥️ **Fullscreen Mode**
Explore the entire space scene in fullscreen.
